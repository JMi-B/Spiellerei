#vokabeln 

| Vokabel   | Bedeutung                   | Erklärung                                                                                                                                                               |
| --------- | --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| uri       | Uniform Resource Identifier | ein allgemeiner Begriff für eine Zeichenfolge um eine abstrakte oder physische Ressource zu identifizieren. zB URL oder ISBN                                            |
| https://  | Protokoll                   |                                                                                                                                                                         |
| www       | Subdomain                   |                                                                                                                                                                         |
| testseite | Domain                      |                                                                                                                                                                         |
| de        | Top Level Domain            |                                                                                                                                                                         |
| def       | Definition                  | Definiert eine Funkiton oder auch Methode                                                                                                                               |
| sip       | Session Identifier Protocol | Identifier der gesetzt wird um eine Session zu identifizieren, wenn das Protokoll (z.B. http ) keine ID vergibt. Wird in der Swxyware für die einzelnen Rufe verwendet. |
