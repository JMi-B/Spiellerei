# Aufgaben
## Passwort-Generator
### Beschreibung
- Eingabe Aufforderung an den Nutzer eine Zahl von 4 - 12 einzugeben
	- Überprüfung der Länge mit einer if Abfrage
- die Eingabe bestimmt wie oft die Schleife durchlaufen wird
- Aus den 4 Zeichenkategorien wird zufällig eine ausgewählt
	- Kennzeichnung der Kategorie als true
	- aus der Kategorie wird zufällig ein Zeichen ausgewählt.
	- das Zeichen wird an den PW string angefügt
- Nach allen Durchläufen wird geprüft, ob die Zeichenkategoriemarker alle true sind
	- wenn nur eins false ist wird die Schleife neu gestartet
	- alle true Passwort ausgeben
### Pseudocode
input 4 -12
if 4<>12 Fehler
	for Dauer <= Input
		random Auswahl aus Korb 1,2,3, oder 4
		 Korb = true
			 rendom Auswahl Zeichen aus Korb
			 Zeichen zum PWString hinzufügen
			 Dauer +1
	 wenn alle Körbe  = true
		 Ausgabe des PW
		 sonst Neustart der for-Schleife continue
			 Dauer = 0
			 Körbe = false
#### Korbverarbeitung
für den mit dem Index ausgewählten Korb
	Kategorie = true
	Länge der Kategorie
		random Zeichen(1, LängeKategorie) aus der Kategorie auswählen
		Zechen an Passwort anfügen
		Dauer +1
