## Hashtag
Dabei geht es darum, aus einem Text einen Hashtag zu machen, den man bei Social Media verwenden kann. Z.B. 

"Das ist super" => "#DasIstSuper" Jeder Hashtag mit muss mit einem # anfangen. Danach sollen aus dem gegebenen Satz bei allen Wörtern der erste Buchstabe groß geschrieben werden, der Rest klein. Und alle Leerzeichen werden entfernt.
### Beschreibung
Ein Sting aus mehreren Wörtern wird bearbeitet und der veränderte String wird ausgegeben.
Die Veränderung hat mehrere Schritte.
- jedes Wort am Anfang groß.
- die Leerzeichen löschen
- ein `#` davor setzen
#### Wie mache ich das:
- in einer SChleife Wortanfang finden ersten Buchstaben `uppercase`
	- Neuer Text
- eine Schleife löscht Leerzeichen
	- Neuer Text
- `#` vorne an den neuen Text setzen 
	- oder das ganze mit einem Hash starten
### Was brauche ich?
- Einen Rahmen der mir Leerzeichen und nächsten Buchstaben sucht
	- DNA ![[AlgorithmusSequenzDNA.py]]
	- Buchstaben `uppercase`
- zweiter Durchlauf `leerzeichen` Löschen![[BuchstabenZahlen.py]]
	-   Hashtag zusammen fügen

Es gibt dafür auch Methoden aber besser zufuss für das Prinzip

## Hangman
### Beschreibung
Ein Galgenmännchenspiel programmieren.
1. Spiel ans Laufen bekommen
2. Liste mit Wörtern, aus denen zu fällig eins ausgelesen wird.
#### Spielablauf
Der Speile bekommt ein Reihe von Platzhaltern für jeden Buchstaben gezeigt.
Der Spieler wird aufgefordert einen Buchstaben einzugeben.
Der eingegebene Buchstabe wird vom Rechener mit dem Wort verglichen.
Wenn der Buchstabe vorkommt wird der neue Sting eingeblendet und eine neue Eingabe erfragt.
WEnn der Buchstabe nicht vorkommt wird von 7 runter gezählt, der String bleibt unverändert und es wird eine neue Eingabe erfragt.
Spiel Endet wenn der Spieler das Wort erraten hat oder der Counter bei Null angekommen ist.
#### Wie mache ich das:
Es gibt einen AusgangsString mit dem Wort.
Es gibt einen AusgabeString:
Der besteht aus Platzhaltern `*` für die Buchstaben `len()`
Schleife
Der `input()` wird mit dem Ausgangsstring verglichen
der Ausgabestring wird neu aufgebaut
- Das vorhandene Zeichen wird beibehalten. Es kann Buchstabe aus der vorherigen Runde sein.
	- `*` wenn der Buchstabe nicht gefunden wurde
- der Buchstabe  wenn er gefunden wurde
der Zähler wird runtergezählt
Ausgabestring wird angezeigt und neue Eingabe erwartet.
Groß und Kleinschreibung spielt keine Rolle.
