# Java Grundlagen
#JAVA
- Java ist ein Compilersprache.
- Klassen und Dateinamen sind immer gleich.
- eine Klasse beginnt mit `public class NAME`
- Jede Klasse beginnt mit `public static void main()`
- Variablen müssen einen Typ haben.
- Was aufgemacht wird muss auch wieder geschlossen werden. `{}`
- Am Ende eines Befehls steht ein `;`
- Mehrere zusammengehörige `classen` ergeben ein `package` ähnlich einem Ordner
- Java ist Casesensetive
- Klassennamen beginnen mit Großbuchstaben

## Schleifen
#Schleife #for
### for
- begrenzte Anzahl an Durchläufen
- syntax `for (statment 1; statment 2; statment3)
	- statment1 - Variabeln definition -Startbedingung wird einmal zu Beginn ausgeführt
	- statment2 -Abbruchbedingung Kondition für die Durchführung 
	- statment3 -wird am ende jeder Schleife ausgeführt.
		- lauf Variable
### for-each
#for-each
- wird für die Elemente eines `array` genutzt
- das Array muss vordem Beginn der Schleife definiert sein
- es werden zwei Variablen benutzt (TeildesArray : Array)
### while
#while 
- while Schleifen werden ausgeführt so lange eine Bedinung `true` ist
- `while (condition) {code}`
	- der Counter steht am Ende des Codes
	- Variablen vorher definieren
### do while
#fussgesteuerteSchleife
- Diese Schleife wird immer einmal ausgeführt. Die Bedingung wird am Ende des Codeblocks geprüft.
- `do { codeblock } while (contition) ;`
- Der Counter steht wieder mit im Codeblock
- Auch wen die Bedindung `false` ist wird der Code einmal ausgeführt.
## Abfragen
### if else
#ifelse
- condition muss `true` oder `false` sein
	- oft mit Vergeleichsoperatoren
	- if Block: wenn eine Bedingung true ist
		- `if (condition){codeblock}`
		- `boolean` Variablen können ohne Operator in die `condition` geschrieben weren
			- `if(isLightOn == true)`ist das gleiche wie `if(isLighton)`
	- else Block: wenn die gliche Bedingung false ist
		- `if(condition)`
			- `{codeblock}`
		- `else` 
			- `{anderer CodeBlock}`
	- else if Block: eine neue Bedingung wenn die erste Bedingung falsch ist.
	`if (Condition)`
		`{codeblock}`
	`else if `
		`{ codeblock}`
	`else`
		`{codeblock}`
	- Es gibt auch eine verkürzte Schreibweise.
	- `variable = (condition)? expressionTrue : expressionFalse;`
	- Für kurze Auswahlen z.B auch in `System.out.println()`
### switch
#switchcase #matchcase
- Wie bei Match case in Python gibt es in Java `switch case` um einzelne Ergebnisse Fälle zu erreichen
- Nach der Code für einen bestimmten Fall ausgeführt wurde wird das Statment abgebrochen
- der Ausdruck wird nur einmal ermittelt und mit den WErten `case values ` verglichen
	`switch (expression){`
		`case x:`
			`codeblock`
			`break;`
		`case y:`
			`codeblock`
			`break;`
		`default:`
		`}`
