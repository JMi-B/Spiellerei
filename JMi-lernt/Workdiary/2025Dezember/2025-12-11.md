# constructor
#JAVA #constructor #methods #classes 
## Erklärung
- Ein Konstruktor ist eine spezielle Methode um ein Objekt zu initialisieren.
	- wenn ein Objekt mit `new Klassenname()` erzeugt wird ruft `()` den Konstruktor auf.
- Die Aufgabe des Konstruktors ist, dafür zu sorgen, das ein Objekt sinnvolle Startparameter hat.
	- Er wird vor allen anderen Methoden als erstes ausgeführt.
- Der `constructor` wird aufgerufen, wenn ein Objekt erstellt wird.
- Er kann dafür benutzt werden um Startwerte für die Attribute des Objekts festzulegen.
- z.B. wenn eine Klasse Main erstellt wird, wird dem Attribut `int x;` mit dem `class constructor` der Start wert `x=5;` fest gelegt.
### Regeln
- Namen von Konstruktoren müssen einen zum Namen der Kasse passen.
- Konstruktoren haben keinen `return type` - wie `void`
- Alle Klassen haben Konstruktoren. Java erstellt selbstständig einen Konstruktor, wenn keiner angegeben wurde.
- Ohne eigenen Constructor können keine Startwerte angelegt werden.
- Wenn ein Konstruktor mit Standartwert angelegt wurde, muss das Attribut in allen Subklassen genutzt werden.
### Parameter
- Konstruktoren können Parameter haben. Die Anzahl ist unbegrenzt.
```java
public class Main {
  int modelYear;
  String modelName;

  public Main(int year, String name) {
    modelYear = year;
    modelName = name;
  }

  public static void main(String[] args) {
    Main myCar = new Main(1969, "Mustang");
    System.out.println(myCar.modelYear + " " + myCar.modelName);
  }
}

// Outputs 1969 Mustang
```
