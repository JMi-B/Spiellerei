# Spiel: Wort aus zufälligen Buchstaben bilden
#pbi #workflow #algorithm 
## Beschreibung
Es wird ein Spiel programmiert. Das Spiel ist ein Wortspiel, bei dem der Spieler aus einer Reihe von zufälligen Buchstaben ein möglichst langes, englisches Wort bildet.

Der Spieler hat nur einen Versuch.
Es werden nur Großbuchstaben benutzt.
- die Umwandlung erfolgt im Programm

Das Programm erstellte einen String aus 9 zufälligen Buchstaben
- 4 Vokale und 5 Konsonanten
- es dürfen Buchstaben doppelt vorkommen

Der Spieler gibt ein möglichst langes Wort aus den gegeben Buchstaben ein.
- Er darf keine anderen Buchstaben verwenden.
- Er darf jeden Buchstaben nur einmal verwenden, bzw. jeden Buchstaben nur so oft verwenden, wie er im String vorkommt

Das Programm prüft ob die Eingaben den Regeln entspricht
- Sind in der Eingabe mehr oder andere Buchstaben als im String, hat der Spieler verloren.
- Ist die Eingabe regelgerecht, erhält der Spieler Punkte.
	- für jeden verwendeten Buchstaben erhält er einen Punkt
	- Wenn er alle Buchstaben verwendet hat, erhält er die doppelte Punktzahl. d.h. 18 Punkte


In diesem ersten Programm für das Spiel, wird nicht geprüft, ob die Eingabe ein sinnvolles englisches Wort ist.

## Akzeptanzkriterien
- Der String wird nach den Vorgaben erstellt und angezeigt.
- Die Prüfung des eingegeben Worts erfolgt nach den oben beschriebenen Regeln.
- Dem Benutzer wird kurz und verständlich erklärt, was seine Aufgabe ist.
- Die Punkte werden richtig berechnet und angezeigt.
- Eingabe von Zahlen oder Sonderzeichen wird nicht akzeptiert.
